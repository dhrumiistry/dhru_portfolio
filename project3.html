Work Project 1 - Manufacturing Data Analysis

This project focused gathering data from production operations & gathering inights from it.

The goal was to identify how many sensors failed weekly & conditions where it is most likely to fail..

A VBA macro was created to clean the data as during a week around 300 - 500 raw data files are created.

This macro gathers key data & removes unneeded columns whilst also fetching data from the summary file.

The data/results then get visualised into a Power BI report for stakeholders to view.

Key Insights

  • Three distinct customer clusters were identified based on frequency and spending.
  • High-value customers showed strong loyalty and responded well to personalized offers.
  • Low-frequency shoppers preferred discounts and seasonal promotions.

Tools Used

- Excel (data gathering)

- VBA (macro)

- Power BI (visualisations & DAX)

Work Project 2 - Testing Limit Change

This project focused around a proposal of changing the limits within 1 of the testing operations.

The goal was to see if tweeking the set limits will positive or negatively impact the amount of sensor failures.

This was done within Power BI using the cleaned data from the previous projects, dropping irrelevant columns however.

Created DAX measures to allow for the scatter chart to become dynamic.
